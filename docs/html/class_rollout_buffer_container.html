Detailed Description

The class RolloutBufferContainer is the main backend container that is used in C_RolloutBuffer. This container stores the data and tracks the references with RolloutBufferData objects to keep transitions and policy outputs seperated. It also provides necessary methods to manipulate the buffer and get its properties.
