◆ extend_transitions()

None rlpack._C.rollout_buffer.RolloutBuffer.extend_transitions (   self)

Method to extend the transitions.

This method will perform gather and extend transition with transitions from other RolloutBuffer instances from other process. This is done synchronously and hence will be a blocking call.

The transitions are only extended for master process and this method will raise an error if called from non-distributed setting.
