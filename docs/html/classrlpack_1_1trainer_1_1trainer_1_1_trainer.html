Detailed Description

This class is a generic class to train or evaluate any agent in any environment.

This class provides necessary framework to perform experimentation and analyse the results. It inherits from rlpack.utils.base.TrainerBase.

Parameters
modebool: Current mode of operation of Trainer (training/evaluation).
agentAgent: The RLPack Agent to be trained or evaluated.
envgym.Env: The gym environment to be used for training or evaluated.
save_pathstr: The path where agent and logs are saved.
num_episodesint: The number of episodes to run for training or evaluation.
max_timestepsUnion[str, None]: The maximum number of timesteps to be run on the environment. This will override the gym environment's TimeWrapper if passed. Default: None.
custom_suffixstr: The custom suffix to add to agent's name while saving for the agent that receives the highest cumulative rewards observed thus far.
reshape_funcOptional[Callable[[np.ndarray, Tuple[int, ...]], np.ndarray]]: The function to reshape the input states. Default: None; Default behavior is to not do any reshaping unless new_shape is passed.
new_shapeUnion[Tuple[int, ...], None]: The new shape for observations (states) using reshape_func. If reshape_func was passed None and a valid new_shape was passed, default reshape func TrainerBase._reshape_func_default is used to perform reshaping.
is_distributedbool: Flag indicating if current setting is distributed or not. If set to True and is not distributed setting (i.e. dist.init_process_group) has not been called yet, may raise an error. Default: False
summary_writerSummaryWriter: The instance of SummaryWriter to perform tensorboard logging. Default: None
