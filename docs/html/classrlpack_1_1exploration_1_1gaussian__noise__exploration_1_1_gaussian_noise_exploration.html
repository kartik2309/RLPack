anneal_funcUnion[GaussianAnnealFuncSignature, None]: Annealing function for loc, scale and weights. Refer to signature in rlpack.exploration.utils.typing_hints.GaussianAnnealFuncSignature. This is an optional parameter and if not passed, no annealing takes place. Default: None
weightThe factor multiply on sampled noise. By default, will create a tensor of ones. Default: None
