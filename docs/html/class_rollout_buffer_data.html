Detailed Description

The class RolloutBufferData keeps the reference of each tensor of RolloutBufferContainer and provides necessary methods to ship tensors to Python.

Member Function Documentation

◆ get_policy_output_data()

std::map< std::string, torch::Tensor > RolloutBufferData::get_policy_output_data ( ) const

Gets the policy output data of the current instance. Must have valid pointers set to action_log_probability, state_current_value and entropy. If valid pointers are not set, an error is raised. This method does not apply tensor options to the tensors.

Returns
The map of policy outputs.

◆ get_transition_data()

std::map< std::string, torch::Tensor > RolloutBufferData::get_transition_data ( torch::TensorOptions &  tensorOptions) const

Gets the transition data of the current instance. Must have valid pointers set to state_current, rewards and dones. If valid pointers are not set, an error is raised. This method applies tensor options to the tensors.

Parameters
tensorOptions: The tensor options to be applied to transition quantities.
Returns
The map of transitions outputs.
